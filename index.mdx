## What is Mintlify?

Mintlify is a modern documentation platform that transforms how you create and maintain documentation. Built with developers in mind, it combines the simplicity of Markdown with powerful features to create documentation your users will love.

## Build powerful documentation

Everything you need to create world-class documentation for your product.

<AccordionGroup>
  <Accordion title="Write with MDX" icon="pen-fancy">
    Use Markdown with MDX components to create rich, interactive documentation. Add custom components, embed code samples, and style your content with minimal effort.

    **Learn more:** [Markdown essentials](/essentials/markdown)
  </Accordion>

  <Accordion title="Code samples" icon="code">
    Display syntax-highlighted code blocks with optional line numbers, file names, and multiple language support. Use CodeGroup for side-by-side comparisons.

    ```bash
    npm i -g mint
    mint dev
    ```

    **Learn more:** [Code blocks](/essentials/code)
  </Accordion>

  <Accordion title="API documentation" icon="terminal">
    Auto-generate beautiful API documentation from OpenAPI (Swagger) specifications. Keep your API docs in sync with your codebase effortlessly.

    **Learn more:** [API reference](/api-reference/introduction)
  </Accordion>

  <Accordion title="Custom navigation" icon="map">
    Organize your docs with tabs, groups, and nested pages. Create an intuitive structure that helps users find exactly what they need.

    **Learn more:** [Navigation setup](/essentials/navigation)
  </Accordion>
</AccordionGroup>
