Mintlify provides powerful code block features including syntax highlighting, line highlighting, file names, and multi-language code groups.

## Inline code

Wrap text in backticks to format it as `inline code`:

```md
Wrap text in backticks to format it as `inline code`
```

<Tip>
Use inline code for file names, variable names, commands, and short code snippets within paragraphs.
</Tip>

## Basic code blocks

Create code blocks by wrapping code in triple backticks with a language identifier:

```javascript
const greeting = "Hello, World!";
console.log(greeting);
```

````md
```javascript
const greeting = "Hello, World!";
console.log(greeting);
```
````

### With file names

Add a file name after the language identifier:

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

````md
```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
````

## Code block features

Mintlify supports advanced code block features through meta options:

### Line highlighting

Highlight specific lines to draw attention to important code:

```python {2,4-6}
def calculate_total(items):
    total = 0  # Highlighted line
    for item in items:
        # These lines are highlighted
        price = item.get('price', 0)
        total += price
    return total
```

````md
```python {2,4-6}
def calculate_total(items):
    total = 0  # Highlighted line
    for item in items:
        # These lines are highlighted
        price = item.get('price', 0)
        total += price
    return total
```
````

### Line numbers

Show line numbers for longer code blocks:

```typescript showLineNumbers
interface User {
  id: number;
  name: string;
  email: string;
}

function getUser(id: number): User {
  // Fetch user logic
  return { id, name: "John", email: "john@example.com" };
}
```

````md
```typescript showLineNumbers
interface User {
  id: number;
  name: string;
  email: string;
}
```
````

## Diff highlighting

Show code changes with `+` and `-` prefixes:

```diff
function example() {
-  const oldValue = "deprecated";
+  const newValue = "updated";
   return newValue;
}
```

````md
```diff
function example() {
-  const oldValue = "deprecated";
+  const newValue = "updated";
   return newValue;
}
```
````

<Note>
Diff blocks are useful for showing before/after comparisons in migration guides and changelog documentation.
</Note>

## CodeGroup for multiple languages

Use `<CodeGroup>` to display code examples in multiple languages with tabs:

<CodeGroup>

```javascript JavaScript
const response = await fetch('https://api.example.com/data');
const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get('https://api.example.com/data')
data = response.json()
print(data)
```

```bash cURL
curl -X GET 'https://api.example.com/data' \
  -H 'Content-Type: application/json'
```

```go Go
resp, err := http.Get("https://api.example.com/data")
if err != nil {
    log.Fatal(err)
}
defer resp.Body.Close()
```

</CodeGroup>


## Advanced features

### Title attribute

Add a custom title to your code block:

```json title="config.json"
{
  "name": "my-app",
  "version": "1.0.0",
  "dependencies": {
    "react": "^18.0.0"
  }
}
```

### Word wrap

Enable word wrapping for long lines:

```text wrap
This is a very long line of text that would normally extend beyond the visible area, but with word wrapping enabled, it will automatically wrap to the next line for better readability.
```

### Collapsible code blocks

Make code blocks expandable to save space:

```javascript expandable
// This code block can be collapsed
function veryLongFunction() {
  const step1 = performOperation();
  const step2 = processData(step1);
  const step3 = validateResult(step2);
  const step4 = formatOutput(step3);
  return step4;
}
```
