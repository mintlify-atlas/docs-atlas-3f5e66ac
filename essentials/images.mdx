Enhance your documentation with images, videos, and other media elements. Mintlify supports multiple ways to embed media with customization options.

## Images

### Markdown syntax

Use standard markdown syntax to add images:

```md
![Image description](/images/screenshot.png)
```

<Note>
Store images in the `/images` directory at the root of your docs project. The image file size must be less than 5MB.
</Note>

### HTML img tag

For more control over image display, use HTML:

```html
<img src="/images/screenshot.png" alt="Description" />
```

You can add styling attributes:

```html
<img 
  src="/images/screenshot.png" 
  alt="Description"
  height="200"
  style={{ borderRadius: '0.5rem' }}
/>
```

### External images

For images larger than 5MB, host them externally and reference the URL:

```md
![Large image](https://cdn.example.com/large-image.jpg)
```

<Tip>
Consider using CDN services like [Cloudinary](https://cloudinary.com/) or [AWS S3](https://aws.amazon.com/s3/) for hosting large images.
</Tip>

## Videos

### YouTube embeds

Embed YouTube videos using an iframe:

<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/4KzFe50RQkQ"
  title="YouTube video player"
  frameBorder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
  style={{ width: '100%', borderRadius: '0.5rem' }}
></iframe>

```html
<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/VIDEO_ID"
  title="YouTube video player"
  frameBorder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
  style={{ width: '100%', borderRadius: '0.5rem' }}
></iframe>
```

<Tip>
Replace `VIDEO_ID` with the actual YouTube video ID from the URL.
</Tip>

### Vimeo embeds

Embed Vimeo videos similarly:

```html
<iframe
  src="https://player.vimeo.com/video/VIDEO_ID"
  width="640"
  height="360"
  frameBorder="0"
  allow="autoplay; fullscreen; picture-in-picture"
  allowFullScreen
  style={{ width: '100%', borderRadius: '0.5rem' }}
></iframe>
```

### Loom embeds

Embed Loom videos for tutorials and walkthroughs:

```html
<iframe
  src="https://www.loom.com/embed/VIDEO_ID"
  frameBorder="0"
  allowFullScreen
  style={{ width: '100%', height: '400px', borderRadius: '0.5rem' }}
></iframe>
```

## HTML embeds

<Tip>
Mintlify supports [HTML in Markdown](https://www.markdownguide.org/basic-syntax/#html), giving you infinite flexibility for custom embeds and layouts.
</Tip>

### Interactive embeds

Embed interactive content like CodeSandbox, Figma, or other tools:

```html
<iframe
  src="https://codesandbox.io/embed/PROJECT_ID"
  style={{ width: '100%', height: '500px', border: 0, borderRadius: '0.5rem' }}
  allow="accelerometer; ambient-light-sensor; camera; encrypted-media; geolocation; gyroscope; hid; microphone; midi; payment; usb; vr; xr-spatial-tracking"
  sandbox="allow-forms allow-modals allow-popups allow-presentation allow-same-origin allow-scripts"
></iframe>
```

## Responsive images

Make images responsive using CSS:

```html
<img 
  src="/images/screenshot.png" 
  alt="Responsive image"
  style={{ width: '100%', maxWidth: '600px', height: 'auto' }}
/>
```

## Dark mode images

Provide different images for light and dark themes:

```html
<img 
  src="/images/logo-light.png" 
  alt="Logo"
  className="block dark:hidden"
/>
<img 
  src="/images/logo-dark.png" 
  alt="Logo"
  className="hidden dark:block"
/>
```

<Warning>
Always test your images in both light and dark modes to ensure good visibility and contrast.
</Warning>
