Mintlify documentation is built with MDX, an extension of Markdown that supports JSX components. You can use standard markdown syntax alongside powerful Mintlify components to create rich, interactive documentation.

## Frontmatter

Every MDX page starts with YAML frontmatter that defines metadata:

```yaml
---
title: 'Page title'
description: 'Page description for SEO'
icon: 'icon-name'
---
```

<Note>
The `title` and `description` fields are required. The `icon` field is optional and uses [Font Awesome](https://fontawesome.com/icons) icon names.
</Note>

## Headings

Use heading levels to structure your content. Each heading automatically creates an anchor link and appears in the table of contents.

```md
## Main section heading

### Subsection heading

#### Minor heading
```

<Tip>
Each **heading** creates an anchor and shows up in the table of contents on the right side of the page.
</Tip>

## Text formatting

Mintlify supports all standard markdown formatting options:

| Style         | Syntax            | Result          |
| ------------- | ----------------- | --------------- |
| Bold          | `**bold**`        | **bold**        |
| Italic        | `_italic_`        | _italic_        |
| Strikethrough | `~strikethrough~` | ~strikethrough~ |

You can combine formatting styles: `**_bold and italic_**` produces **_bold and italic_** text.

### Superscript and subscript

Use HTML tags for superscript and subscript text:

| Style       | Syntax                   | Result                 |
| ----------- | ------------------------ | ---------------------- |
| Superscript | `<sup>superscript</sup>` | <sup>superscript</sup> |
| Subscript   | `<sub>subscript</sub>`   | <sub>subscript</sub>   |

## Lists

### Unordered lists

Create bullet lists with `-`, `*`, or `+`:

```md
- First item
- Second item
- Third item
```

### Ordered lists

Create numbered lists with `1.`, `2.`, etc.:

```md
1. First step
2. Second step
3. Third step
```

### Nested lists

Indent items with spaces to create nested lists:

```md
- Parent item
  - Nested item
  - Another nested item
- Another parent item
```

## Links

### External links

Create links by wrapping text in `[]` followed by the URL in `()`:

```md
[Link to Google](https://google.com)
```

### Internal links

Links to pages in your docs should be root-relative paths:

```md
[Link to another page](/essentials/code)
```

<Warning>
Avoid relative links like `../code`. Root-relative links load faster because Mintlify can optimize them.
</Warning>

## Blockquotes

### Single-line blockquotes

Add a `>` before a paragraph:

> Dorothy followed her through many of the beautiful rooms in her castle.

```md
> Dorothy followed her through many of the beautiful rooms in her castle.
```

### Multi-line blockquotes

Use `>` on each line:

> Dorothy followed her through many of the beautiful rooms in her castle.
>
> The Witch bade her clean the pots and kettles and sweep the floor and keep the fire fed with wood.

```md
> Dorothy followed her through many of the beautiful rooms in her castle.
>
> The Witch bade her clean the pots and kettles and sweep the floor and keep the fire fed with wood.
```

## Tables

Create tables using markdown syntax:

```md
| Column 1 | Column 2 | Column 3 |
| -------- | -------- | -------- |
| Row 1    | Data     | Data     |
| Row 2    | Data     | Data     |
```

| Column 1 | Column 2 | Column 3 |

## LaTeX

Mintlify supports mathematical expressions using the `<Latex>` component:

<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>

```mdx
<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>
```
